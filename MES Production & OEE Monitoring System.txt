# MES Production & OEE Monitoring System

## Project Overview

This project is a simple Manufacturing Execution System (MES) simulation developed in Python. The goal of the project is to understand how production data can be collected and used to calculate Overall Equipment Effectiveness (OEE), which is a key performance indicator in manufacturing industries.

The project uses Object-Oriented Programming (OOP) concepts to model production data and OEE calculations.

## What This Project Does

- Tracks production count for a machine
- Calculates Availability
- Calculates Performance
- Calculates Quality
- Calculates Overall Equipment Effectiveness (OEE)
- Demonstrates the use of Python classes and objects in a manufacturing-related application

## OEE Calculation

The OEE value is calculated using the following components:

- Availability = Runtime / Total Time
- Performance = (Cycle Time × Total Count) / Runtime
- Quality = Good Count / Total Count
- OEE = Availability × Performance × Quality
