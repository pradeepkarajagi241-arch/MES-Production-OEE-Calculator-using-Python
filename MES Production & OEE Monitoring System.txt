MES Production & OEE Monitoring System
Overview

This project is a basic Manufacturing Execution System (MES) simulation developed using Python. It demonstrates production tracking and Overall Equipment Effectiveness (OEE) calculation using Object-Oriented Programming (OOP) concepts.

Features
Production count tracking
Availability calculation
Performance calculation
Quality calculation
Overall Equipment Effectiveness (OEE) calculation
Python OOP implementation
Technologies Used
Python
Object-Oriented Programming (OOP)
OEE Formula
Availability = Runtime / Total Time

Performance = (Cycle Time × Total Count) / Runtime

Quality = Good Count / Total Count

OEE = Availability × Performance × Quality
Project Structure
Production Class
    ├── Machine ID
    ├── Production Count
    └── Total Production

OEE Class
    ├── Availability
    ├── Performance
    ├── Quality
    └── OEE Calculation
